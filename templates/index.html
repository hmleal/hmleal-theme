Configurando envio de e-mails para teste

  • Dom 05 julho 2015

Últimamente tenho trabalhado bastante com envio de e-mails utilizando django e uma das questões que no inicio me encomodou foi como testar o layout sem precisar enviando para minha conta do gmail os e-mails gerados pelo meu sistema.

Foi nesse momento que encontrei uma ferramenta muito interessante que foi o que foi o mailcatcher um servidor smtp que fica escutando uma porta local e capturado todos os emails enviados pelo django.

O único problema foi que esse app é escrito em Ruby, o que não me permitia colocar ele junto com as dependências do projeto. Vagando pelo github foi que encontrei o maildump um clone do mailcatcher só que totalmente escrito em python.
